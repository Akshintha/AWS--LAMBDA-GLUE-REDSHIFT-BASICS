# AWS Basics: Lambda, Glue & Redshift

## AWS Lambda
AWS Lambda is a serverless computing service that allows you to run code without managing servers. It executes your code automatically when an event occurs.

---

## AWS Glue
AWS Glue is an ETL (Extract, Transform, Load) service that helps collect, clean, transform, and move data between different AWS services.

---

## Amazon Redshift
Amazon Redshift is a cloud-based data warehouse used to store large amounts of data and perform fast SQL queries for analytics.

---

# Comparison

| Service         | Purpose                          | Example |

| AWS Lambda      | Runs code automatically          | Process a file when it is uploaded to S3 |
| AWS Glue        | Cleans and transforms data (ETL) | Clean customer data before storing it |
| Amazon Redshift | Stores data for analytics        | Run SQL queries on millions of records |
